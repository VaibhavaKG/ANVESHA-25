# ANVESHA-25
UNSUPERVISED PORTFOLIO OPTIMIZATION
~ A K-Means Clustering Strategy on S&P 500 Stock

## ABSTRACT:
This project presents an unsupervised machine learning–based trading strategy that employs the K-Means clustering algorithm on S&P 500 stock
data to identify hidden market structures and behavioral patterns. Rather than relying on predictive models, the approach emphasizes pattern
recognition and portfolio diversification by grouping stocks with similar historical characteristics, including volatility, momentum, and beta. Clusters
representing distinct risk–return profiles are analyzed to construct optimized portfolios using the Efficient Frontier framework, aiming to maximize the
Sharpe ratio and achieve an optimal balance between return and volatility.
The performance of these cluster-based portfolios is benchmarked against the S&P 500 index to evaluate their effectiveness in enhancing risk-adjusted
returns. Overall, this study demonstrates how integrating unsupervised learning techniques with modern portfolio theory can yield adaptive,
interpretable, and systematically diversified trading strategies in dynamic financial markets.
